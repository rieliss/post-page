# How to run this project

## Installation

Use the package manager package.json file to install package

## On main path

path: project-final-3

```bash
npm install
```

## On client folder

path: project-final-3

```bash
cd client
```

then

```bash
npm install
```

## On server folder

path: project-final-3

```bash
cd server
```

then

```bash
npm install
```

## Usage on client and server command

```bash
npm run start
```
